# Terminal-Arcade
Terminal Arcade é um projeto em Python que reúne mini-jogos jogáveis direto no terminal.

Jogos atuais

- Calculadora
- Pedra, Papel e Tesoura

Objetivo

O projeto foi criado para praticar programação em Python enquanto desenvolvo pequenos jogos e sistemas no terminal.

Tecnologias

- Python 3
- Módulos padrão ("random", "time", "sys")

Como executar

1. Baixe o projeto
2. Abra o terminal na pasta do projeto
3. Execute:

python main.py

Controles

Escolha o número correspondente ao jogo no menu principal.

Exemplo:

1 → Calculadora
2 → Pedra, Papel e Tesoura
0 → Sair
